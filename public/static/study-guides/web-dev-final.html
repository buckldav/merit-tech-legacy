Web Development Study Guide

This study guide will help you refresh the concepts in Web Development that will be on the state test.

The Internet

Vocabulary

Term Definition
The Internet Refers literally to the wires and computers that make up the global network.
Internet Service Provider (ISP) A company that allows you to physically connect to the internet as defined above. Examples: xfinity, CenturyLink, Google Fiber.
Web Servers Special computers on the internet that host websites.
Browser A program (like Google Chrome) that accesses files on the internet from web servers and displays them in a human-readable form.
WiFi A local wireless hotspot allowing access to the internet through the ISP.
Modem The physical device that is connected to your ISP that you connect your computer through WiFi or ethernet.
Router Connects multiple networks together and routes traffic between them.
Packets Chunks of data sent over the internet from web servers to your browser upon request.
IP Address The "telephone number" for a web server. Example: 10.1.1.1
Domain Name
TLD = Top Level Domain
SLD = Second Level Domain
URL = Uniform Resource Locator
Domain Name Server (DNS) Its main job is to translate between Domain Names and IP Addresses.
Internet Protocols
Term Definition
http HyperText Transfer Protocol (http://): The way packets are sent over the web.
https (https://) Same as http, but the packet data is encrypted.
ftp File Transfer Protocol (file://): The way to access a file on a web server.
tcp/ip Transmission Control Protocol/Internet Protocol: The data stream service across the internet.

HTML

Vocabulary

Term Definition
Tag An opening tag for a paragraph is <p> and a closing tag is </p>
Element A component of an HTML document, made up of an opening tag, closing tag, and content.
Nesting Putting an HTML Element inside another element. 
<section>
  <p>This paragraph is nested inside the section</p>
</section>
Attribute A detail associated with an HTML Element, of the format attribute="". Belongs in the opening tag of the element.
Example of anchor tag with an href attribute: <a href="page.html">Page</a>.

The Foundational Structure of a Web Page


  <!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="description" content="Description of the web page for Search Engine Optimization">
    <title>Title of the page (appears in the tab of the browser and on search engines)</title>
    <link href="style.css" rel="stylesheet">
  </head>
  <body>
    <p>Page Content</p>
  </body>
  </html>

Naming, File Structure Conventions

  • The home page should be called index.html.
  • All filenames should be lowercase, with - instead of spaces.
  • That same convention (lowercase, -) is used for ids and classes. Example: id="big-picture".
  • Use lowercase for elements. Example: <p> instead of <P>.

Web Design

When designing a website, you should do the following:

  1. Identify the purpose and audience for the website.
  2. Wireframe the site (sketch out a plan of how the site should look and work).
  3. Maintain consistency in design (use just a few standard fonts and colors).
  4. Make site accessible to screen readers (alt tags in images, strong instead of bold, etc.).
